Location Output Data

Location Output Data shows the Project Aria device‚Äôs position and orientation in the world across six dimensions of freedom.

Each recording has a corresponding trajectory.csv file, which contains the co-location data of the Project Aria device. The trajectory is saved by a series of timestamped transformations of the Project Aria device's left IMU relative to the ‚Äúworld‚Äù frame. The transformations are saved in 1KHz, to give high transformation accuracy when interpolate the trajectory at any arbitrary timestamp.

To give co-location information across different recordings, trajectories from the same ‚Äúlocation‚Äù are all aligned into the same ‚Äúworld‚Äù frame. Trajectories from the same ‚Äúsequence‚Äù are all aligned into the same ‚Äúworld‚Äù frame. The origin of the ‚Äúworld‚Äù frames are arbitrarily defined. The gravity acceleration vector always points up in the z axis of the ‚Äúworld‚Äù frame.
